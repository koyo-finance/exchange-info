## Kōyō Finance V1 Analytics

An open sourced interface for Kōyō Finance V1 analytics - forked from Uniswap V3 analytics.

Info URL: https://info.koyo.finance/#/

## Development

### Install Dependencies

```bash
yarn
```

### Run

```bash
yarn start
```
